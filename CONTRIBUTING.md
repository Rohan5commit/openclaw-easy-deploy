# Contributing

## Requirements

- Node.js 22+
- npm 10+

## Setup

```bash
git clone https://github.com/<your-org>/openclaw-easy-deploy.git
cd openclaw-easy-deploy
npm install
```

## Development workflow

1. Create a branch from `main`.
2. Make focused changes.
3. Run `npm run lint && npm run test && npm run build`.
4. Open a pull request with context and test evidence.

## Commit style

Use clear, imperative commit messages.

## Reporting issues

Use the GitHub issue templates for bugs and platform requests.
